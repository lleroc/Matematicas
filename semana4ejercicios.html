Ejercicios de Matrices

Asignatura: Matemáticas | Carrera: Medicina Veterinaria

Ejercicio 1: Identificación de Tipos de Matrices

Dada la siguiente matriz:

    A = 
    [ 1  0  0 ]
    [ 0  1  0 ]
    [ 0  0  1 ]

Responde:

  • ¿Qué tipo de matriz es?
  • ¿Cuántas filas y columnas tiene?
  • ¿Cuál es el elemento en la posición (2,2)?

Ejercicio 2: Suma de Matrices

Suma las siguientes matrices:

    A = [ 2  4 ]      B = [ 1  3 ]
        [ 6  8 ]          [ 5  7 ]

Escribe la matriz resultante.

Ejercicio 3: Producto de Matrices

Calcula el producto de las siguientes matrices:

    A = [ 1  2 ]      B = [ 3 ]
        [ 3  4 ]          [ 4 ]

Escribe la matriz resultante.

Ejercicio 4: Matriz Insumo-Producto

En una clínica veterinaria, se tienen los siguientes datos:

  • El alimento A cuesta $5 por unidad y el medicamento B cuesta $10 por unidad.
  • Los perros consumen 2 unidades de alimento y 1 de medicamento.
  • Los gatos consumen 1 unidad de alimento y 1 de medicamento.

Representa estos datos en una matriz y calcula el costo total para alimentar a 3 perros y 4 gatos.

Ejercicio 5: Resta de Matrices

Resta las siguientes matrices:

    A = [ 7  5 ]      B = [ 2  3 ]
        [ 4  6 ]          [ 1  4 ]

Escribe la matriz resultante.
